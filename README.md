# Job Listings UI

A responsive job listing card UI built with React and Vite.

## Features
- 10 real FAANG/big tech job listings
- Dynamic props-based card components
- Save/bookmark button
- Responsive grid layout

## Tech Stack
React, Vite, Lucide React, CSS
